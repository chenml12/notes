1.页面三大组成 html---超文本标记语言 css---重叠样式表 js---动态脚本语言 2.js组成 ECMAScript5.1：定义了js的命名规范、变量、数据类型、作语句等基本的知识(js的核心) DOM：document object model 文档对象模型（里面提供了很多的方法和属性来让我们操作页面上的元素） BOM：browser object model 浏览器对象模型（提供了很多用来操作浏览器组件的相关方法和属性） 区分： 文档对象document和文档对象模型DOM 浏览器对象window和浏览器对象模型BOM 3.JS的命名规范 1）JS中严格区分 大小写 2）使用驼峰命名法 1.可以使用字母，数字，下划线(_)，美元符($) 2.第一位不可以是数字 3.不可以使用关键字，保留字 5.不较真的前提下，长度任意 6.必须先声明，在调用 关键字：在js中有特殊意义的字 保留字：未来可能成为关键字的 4.引入方式 （1）css引入方式 行内式 内嵌式（将css样式写在style标签块里面，放在header里面） 外链式（只是将内嵌式中的css样式放到外面一个单独的样式文件中了） 导入式@import("css/index.css") （2）js引入方式 行内式（不推荐，安全性能比较低） 内嵌式（将js代码块写在script脚本块中间) 外链式（将js代码写在外面文件中，通过src找到引入） 优点：方便多个页面复用，扩展，维护，修改 细节： 1）在外链式中，script脚本块中间不可以写js代码，写了也不执行 2)我们通常将js放在body的最里面 原因： html页面是从上到下加载的，js通常是获取html标签给予动态操作效果的，所以需要先加载html标签在加载我们的js 5.变量： 作用：复用数据，代码简洁 js中的变量是一个抽象的概念，变量是用来存储值和代表值的 js中定义一个变量非常的简单：var变量名=变量值 变量：可以存储任何的数据类型 6.JS中常用的几种输出方式： （1）alert（"内容"） 在页面中输出显示我们的内容 （2）document.write("内容")在页面中输出显示我们的内容 （3）console.log("内容")最常用的一种方式在控制台输出我们的内容 （4）innerHTML/innerText:向指定元素中动态添加内容 （5）console.dir("内容")打印详细信息 7.浏览器（window浏览器）----> 文档（document文档对象）---->html----->header/body 8.什么是js：JavaScript是一门轻量级的脚本编程语言 9.初始函数（函数function） 作用：类似变量，复用多行代码 有名函数 声明有名函数 function fn(){ } 直接调用 fn() 事件调用 元素.onclick = 函数名 注意：函数名后边不加括号，不然还没点就执行了 匿名函数 function(){ } 注意：直接声明匿名函数会报错 事件声明 元素.onclick = function(){ } 10.属性操作 操作的是行间属性，可读可写 .的方法 不合符小驼峰命名的属性，样式名，需要转成小驼峰 []的方法 css样式名必须加引号 变量必须用[] 11. +连接符 1.两边都是数字，进行加法运算 2.两边都是字符串，进行字符串拼接 3.只要有一边是字符串，就进行字符串拼接 4.如果是变量，先把变量转成存的数据类型，再按上述方法选择执行 12.数组 arr = [] 作用：存储多个数据 特性：有序性，多个数据间用逗号分隔 取值：数字下标arr[0],从0开始,最后一位length-1 数据个数：arr.length 13.动态获取元素 标签名和class获取元素是动态的 注意：获取时加数字下标就不是动态啦 14.for循环 for(初始化变量;判断条件;条件改变){ 要循环执行的代码 } 1.初始化变量 2.判断条件，当判断条件为真，才执行下边代码，决定了循环执行次数 3.执行代码 4.条件改变 5.判断条件，执行，改变 初始化变量->判断条件->执行->条件改变->判断条件->执行... *注意： 1.初始化只执行了一次 2.判断条件决定了循环是否执行，决定了执行次数 3.每次一定要有条件改变！！！不然就会无限循环，死循环
